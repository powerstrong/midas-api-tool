# MIDAS API Tool TODO

## 제품 방향
- 이 제품은 JSON 편집기가 아니라 MIDAS 입력 도구로 만든다.
- 사용자가 JSON을 직접 작성하거나 수정하게 두지 않는다.
- JSON payload 생성은 프로그램 내부에서 자동 처리한다.
- 목표는 개발자 중심 MIDAS API를 비개발자도 쉽게 쓰게 만드는 것이다.
- 사용자는 엑셀 데이터 복사/붙여넣기와 표 편집만으로 MIDAS DB를 입력할 수 있어야 한다.
- 각 기능은 MIDAS API 매뉴얼과 MIDAS 매뉴얼을 함께 참고해 실무 맥락까지 설명하는 화면으로 만든다.
